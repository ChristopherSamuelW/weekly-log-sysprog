	(intinya process adalah program saat dijalankan). Untuk materi baru, ada beberapa hal yang saya pahami
	dari menonton video dan eksplorasi di internet. Salah satu yang saya pelajari adalah perbedaan system vs.
	application programming, yang intinya adalah hasil app. programming lebih berguna untuk user

	Lalu, saya juga mempelajari soal hierarki directory di Unix OS (terutama Linux). Hierarkinya dimulai dengan

	untuk masalah relative path di hierarki tersebut, yang saya pahami adalah path utuhnya didapat sistem dengan 
	menambahkan current path (yang dapat dilihat dengan command pwd) ditambah dengan relative path yang kita tuliskan,
	yang akhirnya menjadi absolute path juga. Terakhir, saya juga belajar bahwa di Linux, informasi mengenai sistem
	tersimpan di directory /proc (seperti /proc/meminfo untuk informasi mengenai memory, atau /proc/cpuinfo untuk
	info mengenai CPU dan processor).
